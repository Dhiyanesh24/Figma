# Ex09 Event Registration Web Application
## Date: 12/11/23

## AIM:
To design, develop and deploy a web application for event registration.

## DESIGN STEPS:

### Step 1:

### Step 2:

### Step 3:

### Step 4:

### Step 5:
Create pages based on your need and link them.

Publish the website in the given URL.
